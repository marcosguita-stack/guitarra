## 📚 Arquivos Markdown (.md)

**Arquivos a acessar:**
- `CURSO-INTENSIVO-OUVIDORIA-48H.md` (conteúdo principal)
- `SIMULADO-TESTE-PRÁTICO.md` (prática)
- `GUIA-RAPIDO-MEMORIZACAO.md` (flashcards)
- `README.md` (instruções)

### OPÇÃO 1 — Visual Studio Code (RECOMENDADO) ⭐

**Por quê:** Renderiza markdown com cores, estrutura clara, fácil navegação

**Passo 1:** Baixar VS Code (grátis)
```
Acesse: https://code.visualstudio.com
Download a versão para seu OS (Windows/Mac/Linux)
Instale (é rápido)
```

**Passo 2:** Abrir arquivo
```
1. Abra VS Code
2. File → Open File
3. Selecione qualquer arquivo .md
4. Markdown renderizado com cores! ✨
```

---

### OPÇÃO 2 — Bloco de Notas (Simples)

**Por quê:** Já vem no Windows, zero setup

```
1. Right-click no arquivo .md
2. "Abrir com..."
3. Selecione "Bloco de Notas"
4. Arquivo abre (sem cores, mas legível)
```

---

### OPÇÃO 3 — GitHub (Online, ZERO DOWNLOAD) ⭐

**Por quê:** Funciona em qualquer navegador, renderização bonita

**Se os arquivos estão no GitHub:**
```
1. Acesse repositório no GitHub
2. Clique no arquivo .md
3. GitHub renderiza automaticamente
   (estrutura clara, links clicáveis, cores)
```

**Se ainda não está no GitHub:**
```
1. Crie conta grátis: https://github.com
2. Crie repositório novo (private ou public)
3. Upload dos arquivos .md
4. GitHub renderiza automaticamente
```

---

### OPÇÃO 4 — Editor Online Markdown (MAIS FÁCIL)

**Por quê:** Zero instalação, funciona em qualquer navegador

**Ferramenta: Dillinger.io**
```
1. Acesse: https://dillinger.io
2. Copie/cole conteúdo do arquivo .md
3. Renderizado em tempo real! ✨
```

**Alternativas:**
- StackEdit: https://stackedit.io
- HackMD: https://hackmd.io
- Notion: https://notion.so (copie/cole markdown)

---

### OPÇÃO 5 — Converter para PDF (PARA IMPRIMIR)

**Windows:**
```
1. Abra arquivo .md em VS Code
2. Extensão recomendada: "Markdown PDF" (gratuita)
   - Clique Extensions (lado esquerdo)
   - Pesquise "Markdown PDF"
   - Install
3. Right-click no arquivo .md
4. "Markdown PDF: Export (pdf)"
5. Arquivo convertido!
```

**Alternativa Online:**
```
1. Acesse: https://convertio.co/md-pdf/
2. Copie/cole conteúdo do .md
3. Download como PDF
```

## 🎯 Para o Simulado Prático Especificamente
