## 📖 COMO ABRIR ARQUIVOS MARKDOWN (.md)

Se nunca abriu arquivo `.md` antes:

### Opção 1: Bloco de Notas (Mais Simples)
1. Right-click no arquivo `.md`
2. "Abrir com..."
3. Selecione "Bloco de Notas" ou "Notepad"
4. Pronto! Texto abre (sem formatação, mas legível)

### Opção 2: GitHub (Online - RECOMENDADO)
1. Abra GitHub: https://github.com
2. Faça login (ou crie conta grátis)
3. Upload dos arquivos `.md`
4. GitHub formata automaticamente com cores e estrutura

### Opção 3: Visual Studio Code (Profissional)
1. Baixe: https://code.visualstudio.com (grátis)
2. Abra a pasta do projeto
3. Clique no arquivo `.md`
4. Markdown renderizado com cores

### Opção 4: Markdown Editor Online (Nenhum Download)
1. Acesse: https://dillinger.io
2. Copie/cole conteúdo do arquivo `.md`
3. Pronto! Renderizado online

**Minha recomendação:** VS Code (leve, bonito, profissional)
