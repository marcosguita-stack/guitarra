## ⚖️ LGPD — 5 Princípios (MEMORIZE)

```
T - TRANSPARÊNCIA    → Cliente sabe como dados são usados
F - FINALIDADE       → Dados coletados só pro propósito declarado
N - NECESSIDADE      → Coleta mínima, não excesso
S - SEGURANÇA        → Proteção contra vazamento
P - PRESTAÇÃO CONTAS → Documente tudo
```

**Direitos do Cliente (CRÍTICO):**
- ✅ Acessar dados ("quais dados vocês têm?")
- ✅ Corrigir dados errados
- ✅ Deletar (direito ao esquecimento)
- ✅ Portabilidade (levar pra outro lugar)
- ✅ Saber se há tratamento automatizado

**Seu Papel:** Se cliente reclamar de privacidade → Escalopa para DPO em ATÉ 10 DIAS

---

## 🛡️ CDC — 7 Direitos Básicos do Consumidor

```
1. PROTEÇÃO DA VIDA     → Serviço não prejudica saúde
2. EDUCAÇÃO & INFO      → Transparência clara
3. CONTRA PUBLICIDADE   → Sem mentiras
4. PROTEÇÃO CONTRATO    → Cláusulas abusivas = NULAS
5. INDENIZAÇÃO DANO     → Se prejudicado, tem direito
6. ACESSO JUSTIÇA       → Pode reclamar, processar
7. FACILITAÇÃO DEFESA   → Você ajuda cliente se defender
```

**Cláusulas Abusivas (ANULE):**
- ❌ Taxa absurda por reclamação
- ❌ Proibir cliente de reclamar
- ❌ Transferir risco injustamente
- ❌ Limitar direitos unilateralmente

## 📊 PREVIDÊNCIA — RGPS vs LGPS

| Aspecto | RGPS (INSS) | LGPS (Postalis) |
|---------|---|---|
| **Tipo** | Público/Obrigatório | Privado/Voluntário |
| **Benefício** | Aposentadoria básica | Complementa INSS |
| **Mínimo** | Salário mínimo | Conforme contribuição |

**Direitos Contribuinte Postalis:**
- ✅ Consultar saldo (imediato)
- ✅ Resgate (se elegível)
- ✅ Portabilidade (transferência 30d)
- ✅ Benefício vitalício (aposentadoria)
- ✅ Reclamação (até 60 dias prescritório)

## 🚨 PALAVRAS-CHAVE EM OUVIDORIA

| Termo | Significado |
|-------|---|
| **Protocolo** | Número único da reclamação (sempre gere) |
| **SLA** | Service Level Agreement (prazo máximo) |
| **DPO** | Data Protection Officer (privacidade) |
| **Gozo** | Cliente recebendo benefício (não pode resgatetudo) |
| **Portabilidade** | Transferência de saldo pra outro fundo |
| **Rentabilidade** | Retorno do investimento (+/- percentual) |
| **Cláusula Abusiva** | Termo ilegal no contrato (anula) |
| **Compliance** | Conformidade com lei |
| **Auditoria Interna** | Investigação de fraude/desvio |
